# arxiv-daily-reports

Daily arXiv theory report artifacts generated by OpenClaw automation.

## Contents
- `reports/`: generated report files (`.tex`, `.pdf`)

## Source automation
This repo is intended to replace Google Docs/Drive as the storage target for the arXiv daily report cron job.

## Keyword filter used by the cron job
The daily crawler filters papers by matching at least one of the following keywords (title + abstract, case-insensitive), in this priority order:
1. Ginibre
2. non-Hermitian / non Hermitian
3. Wishart
4. random matrix / random matrices
5. spectral
